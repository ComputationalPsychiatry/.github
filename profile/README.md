TAPAS is a collection of software tools intended to support the development of computational assays (Translational Neuromodeling) and their clinical application in Computational Psychiatry, Neurology and Psychosomatics.
For a general overview of the TAPAS toolbox suite, including its mission, please refer to:
- Fr√§ssle, Stefan et al. "TAPAS: An Open-Source Software Package for Translational Neuromodeling and Computational Psychiatry". *Frontiers in Psychiatry* 12 (2021). [https://doi.org/10.3389/fpsyt.2021.680811](https://www.frontiersin.org/journals/psychiatry/articles/10.3389/fpsyt.2021.680811/full).

Currently, TAPAS includes the following packages:

- [ActionModels.jl](https://github.com/ComputationalPsychiatry/ActionModels.jl): A Julia package for behavioural modeling
- [ActiveInference.jl](https://github.com/ComputationalPsychiatry/ActiveInference.jl): Computational modeling of active inference.
- [CEODE](https://github.com/ComputationalPsychiatry/CEODE): Continuous Extension of ODE methods. A toolbox for integration of delay differential equations for DCM for EEG/MEG.
- [genbed](https://github.com/ComputationalPsychiatry/genbed): A Python package for data exploration and classification as part of the generative embedding pipeline.
- [hgf-toolbox](https://github.com/ComputationalPsychiatry/hgf-toolbox): The Hierarchical Gaussian Filter; Bayesian inference on computational processes from observed behaviour.
- [HierarchicalGaussianFiltering.jl](https://github.com/ComputationalPsychiatry/HierarchicalGaussianFiltering.jl): The Julia implementation of the generalised hierarchical Gaussian filter
- [PhysIO](https://github.com/ComputationalPsychiatry/PhysIO): Physiological Noise Correction for fMRI.
- [PyHGF](https://github.com/ComputationalPsychiatry/pyhgf): A neural network library for predictive coding
- [rDCM](https://github.com/ComputationalPsychiatry/rDCM): Regression dynamic causal modeling; efficient inference on whole-brain effective connectivity from fMRI data.
- [RegressionDynamicCausalModeling.jl](https://github.com/ComputationalPsychiatry/RegressionDynamicCausalModeling.jl): The Julia implementation of rDCM.
- [UniQC](https://github.com/ComputationalPsychiatry/UniQC): Unified neuroimaging quality control.
